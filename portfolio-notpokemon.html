Mobile work Not Pokemon

A small Xamarin project

This small app was made while I was looking into both mobile development as well as API usage as part of research my team was doing on both.

This small app uses an API called pokeapi to load information about every pokemon into models which are then displayed to the user. The original goal was going to be to try and make a pokemon team builder and then battle preset players, however it turns out attempting to replicate one of the biggest game franchises as part of a learning experience is slightly too large of a goal. So instead the app functions more so as a pokedex.

The source code for this app is not available on my Github however if you were to ever see me in person I would be more than happy to give you a demo of the app as I used my personal phone to debug it on meaning it is saved on my phone
