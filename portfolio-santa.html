Secret Santa game Santas sEGGway

A work in progress!

This work in progress game is going to be my entry for the itch.io Secret Santa 2021 game jam! Although hopefully the person this game is made for doesn't find this page.

In this game jam we each recieved letters from our giftee with a brief description of themselves and what type of game they would like. My giftee asked for a game like Undertale/Deltarune which are amazing games. As this is currently a WIP there's sadly not too much for me to show as of yet however a lot has been done!

I have decided to go for a Christmas/Egg hybrid theme for the game (this idea came while very sleep deprived) and am developing it using the Unity engine. So far the basic mechanics of the game such as a battle system, dialogue system and movement have all been done meaning all that is left is to make more nifty MsPaint art and piece everything together.

For my own unique take on the battles there are only 2 options, ATTACK or HEAL. If the player heals they will recieve a multiplier on how much damage they take over the rest of the battle meaning that healing will also come with a risk.
